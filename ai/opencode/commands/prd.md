# PRD Command

This command triggers the PRD skill to generate a detailed Product Requirements Document.

## Usage

Run this command when you need to create a PRD for a new feature or project.

## What it does

1. Asks clarifying questions about the feature
2. Generates a structured PRD with user stories, acceptance criteria, and requirements
3. Saves the PRD to `tasks/prd-[feature-name].md`

## Trigger phrases

- create a prd
- write prd for
- plan this feature
- requirements for
- spec out
